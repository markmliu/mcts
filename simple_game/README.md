### Usage

## Running the main runner

`g++ runner.cpp tic-tac-toe.cpp --std=c++17`

## Running unit tests

`g++ tic-tac-toe.cpp catch_amalgamated.cpp test_basic_tic_tac_toe.cpp --std=c++17`
